A beautiful, pastel-themed one-page website celebrating why unicorns are the most magical fantasy creatures.

## Features

- Built with Next.js 16 and React 19
- Styled with Tailwind CSS
- Responsive design with pastel colors and unicorn vibes
- Optimized for Vercel deployment

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To create a production build:

```bash
npm run build
```

## Deploy to Vercel

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new).

Simply push this repository to your GitHub account (AloaAgency) and import it to Vercel.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
